# Lulicun-Cars

GetStarted

1. Install Virtualbox
https://www.virtualbox.org/wiki/Downloads

2. Install Vagrant
http://www.vagrantup.com/downloads.html

3. Install node and npm
https://nodejs.org/download/

4. Install bower
npm install -g bower

5. Download Front-end libraries 
cd path/to/public
bower install

6. Fire application
cd root/of/application
vagrant up
vagrant provision
